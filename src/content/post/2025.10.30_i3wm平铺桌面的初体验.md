最近在尝试使用平铺桌面，配置了一下感觉体验感还不错

## 1.i3
直接apt install i3即可

## 2.壁纸的配置
当切换到i3wm中时，整个页面黑黑的一坨，
我使用的是Nitrogen配置的壁纸
i3配置加入一个自启动即可
```
exec --no-startup-id nitrogen --restore
```
BTW，壁纸来源是https://artvee.com/

## 音量
如果要使用笔记本上的音量和亮度键来调节，还需要配置一下
音量调节我使用pamixer

i3的配置：
```
#增加音量

bindsym XF86AudioRaiseVolume exec --no-startup-id pamixer -i 10 && notify-send "音量" "$(pamixer --get-volume)%"
 
# 降低音量
bindsym XF86AudioLowerVolume exec --no-startup-id pamixer -d 10 && notify-send "音量" "$(pamixer --get-volume)%"

# 静音
bindsym XF86AudioMute exec --no-startup-id pamixer -t && notify-send "音量" "$(pamixer --get-mute | grep -q true && echo '已静音' || echo '已取消静音')" 
```
 加了一个弹窗，主要起到一个可视化的作用......

## 亮度
亮度使用：brightnessctl

brightnessctl需要sudo权限，
先配置一个免密
```
sudo visudo -f /etc/sudoers.d/brightnessctl
```
添加：
```
finn ALL=(ALL) NOPASSWD: /usr/bin/brightnessctl
```
保存退出，然后再i3的配置文件中写入
```
bindsym XF86MonBrightnessUp exec --no-startup-id "sudo brightnessctl set +5% | grep -oP '\\d+%' | xargs -I {} notify-send '亮度增加' '当前亮度: {}'"

bindsym XF86MonBrightnessDown exec --no-startup-id "sudo brightnessctl set 5%- | grep -oP '\\d+%' | xargs -I {} notify-send '亮度降低' '当前亮度: {}'"
```
